# Images des Personnages

## Où placer les images ?

Placez vos images de personnages Genshin Impact dans ce dossier.

### Format recommandé

- **Format** : PNG ou GIF
- **Nom** : `Nom_Personnage.png` (ex: `Hu_Tao.png`, `Raiden_Shogun.png`)
- **Taille recommandée** : 256x256px minimum
